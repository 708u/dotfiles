# シチュエーション別パターン集

## 商品紹介

### AI的な表現（避けるべき）

```
本製品は最新技術を採用し、高い品質を実現しています。
価格と性能のバランスに優れ、多くのユーザーに支持されています。
```

### 人間らしい表現

```
これ、友達に「何使ってる？」って聞かれたら迷わず答えるやつ。
値段見たとき「え、安くない？」ってなったけど、
使ってみたら納得。いや、むしろ得した気分。
```

### 変換のポイント

- 第三者視点から一人称視点へ
- スペックの羅列から体験談へ
- 「友人に勧める」シチュエーションを想像
- 驚きや発見の感情を入れる

## ブログ記事

### AI的な表現（避けるべき）

```
プログラミング学習において、継続は重要な要素です。
毎日少しずつ取り組むことで、確実にスキルが向上します。
以下に効果的な学習方法を紹介します。
```

### 人間らしい表現

```
プログラミング、何度も挫折した。
「今度こそ」って思うのに、2週間くらいで手が止まる。

でも去年、やっと続いた。
理由は単純で、毎日5分だけって決めたから。
5分なら、どんなに疲れてても「まあいいか」ってなる。

そしたら不思議なことに、5分で終わらなくなった。
```

### 変換のポイント

- 失敗談や挫折経験から始める
- 時系列で物語として語る
- 「発見」の瞬間を描写
- 読者が共感できる感情を入れる

## 企業紹介

### AI的な表現（避けるべき）

```
弊社は2010年に設立され、IT分野でサービスを提供しています。
社員数50名、売上高10億円の成長企業です。
働きやすい環境づくりに力を入れています。
```

### 人間らしい表現

```
オフィスに入ると、まずコーヒーの香りがする。
誰かが淹れたてを作ったんだな、って思う。

ミーティングは長くない。30分で終わらせようとする空気がある。
ダラダラ話すより、さっさと手を動かしたい人が多いから。

10年でここまで来た。まだまだ途中だけど。
```

### 変換のポイント

- 数字やスペックより「空気感」を伝える
- 社員の日常を切り取る
- 五感で感じられる描写を入れる
- 完璧さより「らしさ」を優先

## レビュー・感想

### AI的な表現（避けるべき）

```
この映画は脚本が優れており、俳優の演技も素晴らしいです。
映像美も際立っており、音楽も効果的に使用されています。
総合的に高く評価できる作品です。
```

### 人間らしい表現

```
見終わって、しばらく席を立てなかった。

あのシーン、主人公が何も言わずに振り返るところ。
あれだけで全部わかった気がした。

帰り道、なんかずっとぼんやりしてた。
「いい映画だった」とかじゃなくて、ただ、ぼんやり。
```

### 変換のポイント

- 分析より「体験」を語る
- 印象に残った一場面を具体的に
- 見終わった後の余韻を描写
- 評価より感情の動きを伝える

## 技術解説

### AI的な表現（避けるべき）

```
このAPIは認証にOAuth 2.0を使用します。
まずクライアントIDとシークレットを取得し、
アクセストークンをリクエストします。
```

### 人間らしい表現

```
最初、認証まわりで3時間溶かした。
ドキュメント読んでも「で、結局どうすれば？」ってなる。

結論から言うと、こうすればいい。

（コード例）

なんでこれでいいのか、順番に説明していく。
たぶん同じところでハマる人いると思うから。
```

### 変換のポイント

- 自分がハマったポイントを共有
- 結論を先に、理由を後に
- 読者への語りかけを入れる
- 「なぜ」を説明する

## 表現パターン集

### 文頭バリエーション

AI的な文頭を避け、変化をつける。

- 「この〜は」 → 「正直、」
- 「〜について」 → 「前から気になってたんだけど、」
- 「〜の特徴は」 → 「使ってみて思ったのは、」
- 「まず」 → 「最初に言っておくと、」
- 「また」 → 「あと、これは余談だけど、」

### 語尾バリエーション

同じ語尾の連続を避ける。

```
〜だと思う。（断定回避）
〜かもしれない。（可能性）
〜な気がする。（感覚）
〜っていう。（引用風）
〜なんだよね。（独り言風）
〜だった。（過去形で変化）
```

### 接続のバリエーション

論理的すぎる接続を崩す。

- 「しかし」 → 「でも」
- 「したがって」 → 「だから」
- 「また」 → 「あと」
- 「さらに」 → 「それと」
- 「例えば」 → 「たとえばの話」

---

## 要素の強度調整

### 軽め（フォーマル寄り）

- 構成は維持しつつ、語尾だけ柔らかく
- 1-2箇所に個人的な感想を挿入
- 「です・ます」は残す

### 中程度（バランス型）

- 構成を少し崩す
- 体験談を1段落追加
- 短文と長文を混ぜる

### 強め（カジュアル）

- 構成を大きく崩す
- 脱線や余談を入れる
- 口語表現を多用
