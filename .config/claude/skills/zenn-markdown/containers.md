# コンテナ記法

Zenn固有のコンテナ記法（`:::`で囲むブロック）について。

## メッセージ

補足情報や注意点を目立たせる。

### 通常メッセージ（黄色背景）

```markdown
:::message
ここに補足情報を記載します。
複数行も可能です。
:::
```

### 警告メッセージ（赤背景）

```markdown
:::message alert
重要な警告や注意事項をここに記載します。
:::
```

## アコーディオン（折りたたみ）

長いコンテンツを折りたたんで表示。

```markdown
:::details タイトル
折りたたまれるコンテンツ。
クリックで展開される。
:::
```

### タイトルの注意点

タイトルには以下を含めない:

- `[` や `]`（リンク記法と競合）
- 改行

## ネスト

コンテナをネストする場合、外側に追加のコロンを付ける。

```markdown
::::details 外側タイトル
外側のコンテンツ

:::message
内側のメッセージ
:::

::::
```

3段以上のネストも同様に、外側ほどコロンを多くする。

## 使用例

### コード例とその説明

````markdown
:::message
以下のコードは非推奨です。代わりに新しいAPIを使用してください。
:::

```js
// 非推奨コード
oldFunction();
```

:::message alert
この変更は破壊的変更を含みます。
:::
````

### 補足情報の折りたたみ

```markdown
:::details 詳細な設定オプション
| オプション | 説明 |
|------------|------|
| option1 | 説明1 |
| option2 | 説明2 |
:::
```
